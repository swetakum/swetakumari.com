    Adding Salmon-based Modules to MultiQC

    MultiQC is a tool used for analyzing multiple sequencing samples together. The tool itself has the ability to parse through different types of outputs and generate reports. So, for example, if a folder contains the output of multiple runs of Salmon (a quantification and mapping tool), MultiQC will be able to parse through them, read the log files, determine which tool was run on the samples, filter out particular features and generate a report using them. The purpose of this project is to make modules for MultiQC that are Salmon-specific.
    Python Bioinformatics

    Implementing G-OLA logic in Spark streaming

    More and more data is being generated every second and is being processed, but the processing of such huge data is turning into long batch jobs. To address this problem, we tried a state based approach on streaming data to make it interactive. The approach was based on a mini-batch execution model called G-OLA based on online aggregation(OLA) to support interactive analysis on batch queries. We basically considered each RDD as a mini-batch and stored a state of it which will be merged with the next RDD to output an answer.
    PySpark Big Data Research
